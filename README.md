# Technical Details

Technologies used:
1. Pandas
2. Numpy
3. Scikit-learn
4. seaborn

The predictions are under `1y_predicted.csv` for the 12 month prediction and `48h_predicted.csv` for the 2 day hourly predictions.

The data we used can be found [here](https://en.ilmatieteenlaitos.fi/).

The pre-trained models are found [here](https://drive.google.com/file/d/1bWPBY5VDer-ovkPjNjnD8iFlKVg90C6v/view?usp=sharing).
