# Technical Details

Technologies used:
1. Pandas
2. Numpy
3. Scikit-learn
4. seaborn

The predictions are under `1y_predicted.csv` for the 12 month prediction and `48h_predicted.csv` for the 2 day hourly predictions.

The data we used can be found [here]([url](https://drive.google.com/file/d/1t32jA2cPxvt_7vClpV7w8g65_FExJtri/view?usp=sharing)).

The pre-trained models are `hourly.joblib` and `monthly.joblib`.
